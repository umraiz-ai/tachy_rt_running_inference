## How to set interface environment
Tachy Device를 사용하기 위해서는 아래와 같이 인터페이스에 대한 환경변수를 설정해주어야 합니다. <br>
Tachy Device별로 사용 가능한 인터페이스는 서로 다르며 본인이 사용하고 있는 Device를 확인해야합니다. <br>

``` bash
# Tachy-Shield
export TACHY_INTERFACE="spi:host"
```
